2. # Cancer Awareness & Support

A simple responsive web page built with HTML, CSS and JavaScript for a cancer awareness and support landing page. It includes a hero section, contact form, and an inspiring quotes section.

## Live Demo

3. Features
- Hero section with cancer awareness message and banner image.
- Contact form with Name, Email, and Message fields (front-end only, no backend).
- Responsive layout that works on desktop and mobile screens.
- Inspiring quotes section that shows a random motivational quote using JavaScript.
